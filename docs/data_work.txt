
Trabajo completado:

* Trabajamos con la API de AEMET para obtener estadísticas y predicciones meteorológicas directamente desde el sitio.
* Basándonos en los parámetros de entrada de la aplicación web, creamos una base de datos de usuarios falsos utilizando la biblioteca Faker.
* Realizamos un Análisis Exploratorio de Datos (EDA) en esta base de datos ficticia para proporcionar análisis estadísticos al departamento de marketing. 
  Esta información es relevante al generar campañas publicitarias.
* Producimos resultados ficticios de cuestionarios para estos usuarios, basados en las preguntas del cuestionario proporcionadas por marketing.
* Creamos un modelo predictivo para el rendimiento en el cuestionario para identificar qué usuarios podrían beneficiarse de más sugerencias y notificaciones de la aplicación.
* Implementamos una función de chatbot para interactuar con el usuario en caso de que soliciten más información sobre un tema relacionado con olas de calor. 
  El chatbot se basa en la comunicación con OpenAI y no responderá preguntas que no estén relacionadas con el tema especificado. En ese caso, los usuarios recibirán una indicación para hacer solo preguntas relacionadas con las olas de calor.
* Implementamos el uso de Streamlit para facilitar el intercambio de nuestros hallazgos con otros departamentos.
